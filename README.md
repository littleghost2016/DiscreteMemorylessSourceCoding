# DiscreteMemorylessSourceCoding
本学期进行李老师所授《信息论》的学习，预期进行几种典型的离散无记忆信源编码的代码实现工作。最后会生成一个可执行文件，拖拽任意文件到可执行文件上，即可进行压缩；拖拽已压缩的文件到可执行文件上，即可自动判断压缩类型并解压。
